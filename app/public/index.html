elmer

Behavior-driven development for Elm Html apps.

See the documentation for lots more information.

To accomplish its magic, Elmer needs to run native Javascript code. For this reason, it's currently not possible to install Elmer via package.elm-lang.org. Nevertheless, you can install Elmer by following the steps below. Enjoy!


Getting Started

  1. Install elm-test. 
    npm install -g elm-test@0.18.7
    Note: elmer 3.0.0 requires elm-test version 4.1.0 or greater. Elmer has been tested with version 0.18.7 of the elm-test node-test-runner.
  2. Install elm-github-install
  3. In your elm project directory, run: 
    elm test init
  4. Go into the tests directory and edit elm-package.json to add the dependency for elmer: 
    "dependencies": {
  "brian-watkins/elmer": "3.0.0 <= v < 4.0.0",
  ...
}
  5. In the tests directory, run: 
    elm-install
Now, use elm-test like normal. Any time you need to update or install new test packages, go to your tests directory and run: 
elm-install
