elmer

Behavior-driven development for Elm Html apps.

To accomplish its magic, elmer needs to run native Javascript code. For this reason, it's currently not possible to install elmer via package.elm-lang.org. Nevertheless, you can install elmer by following the steps below. Enjoy!

See the docs for lots more information.


Getting Started

  1. Install elm-test.
  2. Download elmer-package (see below), unzip it, and place it in your path.
  3. In your elm project directory, run: 
    elm test init
  4. Run: 
    cd tests; elmer-package install brian-watkins/elmer
Now, use elm-test like normal. Any time you need to update or install new test packages, go to your tests directory and run: 
elmer-package install
